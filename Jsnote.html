JavaScript note



書寫位置 敘述
script 寫在html文件中

範例:
< body >
< script >

代碼

< / script >

< /body>
/ 寫在js.格式的文件

範例:
< script src = "js.文件路徑" >< /script>
輸出 敘述
alert() 警告框
console.log() 控制台輸出

變量 敘述
var 聲明變量

範例:var num =1 :聲明名稱為num值為1的變量

命名規則

僅能由數字.字母.下線.$組合,不能數字開頭

嚴格區分大小寫
數據 敘述
Number 所有數字
String 字符串
Boolean true表真,false表假
Undefined 變量沒有值,默認為Undefined
Null 表示空
字符串常用 敘述
CharAt() 指定位置的字符
substring() 提取字符串中指定的字符
substr() 指定位置提取指定數目的字符
toLowerCase() 字符串轉為小寫
indexOf 返回指定字符首次出現的索引

運算符號 敘述
+ 1.兩邊為數字,為加法

2.為字串,為連字符
- 減法
* 乘法
/ 除法
% 取餘數
toFixed() 指定小數後面的位數
關係運算符號 敘述
< 小於
> 大於
>= 大於等於
<= 小於等於
== 等於
!= 不等於
=== 全等於
!== 不全等於
! 非
&& 與
|| 或
++ 自增
-- 自減
條件語氣 敘述
if 條件為true的時候,執行if代碼
else 條件為false的時候,執行else代碼
case 在switch中使用,表要匹配每一中情況
default 在switch中使用,表默認,有條件不符合case時
switch 不同條件執行不同狀況
循環語氣 敘述
for 循環執行一段代碼
while 條件為真,循環代碼塊
名稱 敘述
break 退出循環
continue 跳出當前循環,執行下一次
