7.標籤

(1)圖片

程式碼:< img src="" alt="" >

    屬性
  • src:路徑
  • alt:失敗
  • title:鼠標放置時出現的文字
  • width,height:寬,高

(2)音頻

程式碼:< audio src="" controls autoplay loop >

    屬性
  • src:路徑
  • controls:播放控制
  • autoplay:自動播放
  • loop:循環播放

(2)連結

程式碼:< a href="..." target=_self>連結< /a>

    屬性
  • target:頁面打開方式
  • target=_self:當前窗口打開
  • target=_blank:新窗口打開

(4)表單

input

程式碼:< input type="">

    屬性
  • text:文本
    • 程式碼:< input type="text" placeholder="用戶名" name="" id="">
    • 範例:
  • password:密碼
    • 程式碼:< input type="password" placeholder="密碼">
    • 範例:
  • radio:單選
    • 程式碼:
      < input type="radio" name="choose">yes

      < input type="radio" name="choose">no
    • 範例:yes no
  • checkbox:多選
    • 程式碼:
      < input type="checkbox" checked>多選1

      < input type="checkbox" checked>多選2
    • 範例:多選1>yes 多選2
  • file:上傳文件
    • 程式碼:< input type="file" multiple>
    • 範例:
  • submit,reset,button:按鈕
    • 程式碼:
      < input type="submit" value="註冊>"

      < input type="reset" value="重置>"

      < input type="button" value="普通按鈕>"
    • 範例:
